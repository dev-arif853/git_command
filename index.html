💎 Best Practices

"Wisdom from the ancients (developers who survived merge conflicts)"

1. COMMIT OFTEN
Small commits = easy debugging
# ✅ Good workflow
git add feature.js
git commit -m "Add login form"
git add tests/feature.test.js
git commit -m "Add login tests"
2. PULL BEFORE PUSH
Avoid merge conflicts from hell
git pull origin main # Get latest
git push origin main # Send yours
3. BRANCH EVERYTHING
Never develop directly on main!
git checkout -b feature/awesome
git checkout -b fix/that-bug
4. NEVER FORCE PUSH
Career-limiting move on shared branches
# ❌ NEVER ON MAIN!
git push --force origin main

# ✅ Feature branch okay
git push --force-with-lease origin feature/x
THE GOLDEN RULES
5.
Review Before Committing
Run git diff --staged before every commit
6.
Keep Commits Atomic
One commit = one logical change. Not 47 things at once.
7.
Meaningful Branch Names
feature/user-auth not asdf-2-final
8.
Use .gitignore Wisely
Never commit node_modules/, .env, secrets.txt 🙏
9.
Test Before Committing
If tests fail, fix them first! (Novel concept, we know)
10.
Keep Main Stable
Break things in feature branches. Merge only when reviewed.
🚨 Emergency Commands

"When things go horribly wrong"

😱 "I committed to the wrong branch!"
git reset HEAD~ --soft       # Undo commit, keep changes
git stash                    # Save your changes
git checkout correct-branch  # Go to right branch
git stash pop                # Apply your changes
git commit -m "Right branch!"
😱 "I need to undo my last commit!"
git reset HEAD~1 --soft   # Keep changes, undo commit
git reset HEAD~1 --mixed  # Unstage, keep changes
git reset HEAD~1 --hard   # DESTROY EVERYTHING ⚠️
😱 "I accidentally deleted important code!"
git reflog                          # Find the lost commit
git checkout abc123 -- file.js      # Restore specific file
git reset --hard abc123             # Reset to that point
😱 "Help, I'm in a merge conflict!"
# Option 1: Brave it out
git status       # See conflicts
# Fix files manually
git add .
git commit

# Option 2: Escape!
git merge --abort  # Run away! Start over!
